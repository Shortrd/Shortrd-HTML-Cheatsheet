Element or Attribute Name Formatted Tag Description of Tag Use
Document Metadata (Header) <head> Contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.
Title <title> Defines the document's title that is shown in a browser's title bar or a page's tab.
Body <body> Any visible content should be placed within the opening and closing body tags.
Header <header> Represents introductory content, typically a group of introductory or navigational aids. It may contain some heading elements but also a logo, a search form, an author name, and other elements.
Navigation Section <nav> Represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents. Common examples of navigation sections are menus, tables of contents, and indexes.
Main <main> Represents the dominant content of the <body> of a document.
Footer <footer> Represents a footer for its nearest ancestor sectioning content or sectioning root element. A <footer> typically contains information about the author of the section, copyright data or links to related documents.
Generic Section <section> Represents a generic standalone section of a document, which doesn't have a more specific semantic element to represent it. Sections should always have a heading, with very few exceptions.
Article Contents <article> Represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable (e.g., in syndication). Examples include: a forum post, a magazine or newspaper article, or a blog entry, a product card, a user-submitted comment, an interactive widget or gadget, or any other independent item of content.
Aside <aside> Represents a portion of a document whose content is only indirectly related to the document's main content.
Heading and Sub-headings <h1>, <h2> ... <h6> Headings and sub-headings are used to provide titles for sections of content.
Paragraph <p> Represents a paragraph.
Span <span> A generic inline container for phrasing content. It should be used only when no other semantic element is appropriate. Span is similar to the 'div' element, but 'div' is a block-level element whereas 'span' is an inline element.
Division <div> The generic container for flow content. It's used to group content so it can be easily styled using the class or id attributes.
Emphasis <em> For words that have a stressed emphasis compared to surrounding text. Displayed in italic type, but should not be used to apply italic styling. The <em> element represents stress emphasis of its contents.
Strong <strong> Indicates that its contents have strong importance, seriousness, or urgency.
Line Break <br> Produces a line break in text. It is useful for writing a poem or an address, where the division of lines is significant.
Ordered Lists <ol> Represents an ordered list of items â€” typically rendered as a numbered list.
Unordered Lists <ul> Represents an unordered list of items, typically rendered as a bulleted list.
List Items <li> Used to represent an item in a list. It must be contained in a parent element: an ordered list, an unordered list, or a menu.
Image Embed <img> Embeds an image into the document. Inside <img>, the 'src' attribute is required.
Source src Contains the path to the image you want to embed.
Alternate Text alt Holds a text description of the image, which isn't mandatory but is incredibly useful for accessibility.
Figure <figure> Represents self-contained content, potentially with an optional caption.
Figure Caption <figcaption> Represents a caption or legend describing the rest of the contents of its parent <figure> element.
Video Embed <video> Embeds a media player which supports video playback into the document. Inside <video>, the src attribute is required which contains the path to the image you want to embed. Also, inside the opening tag you can add options such as autoplay, controls, width, height, etc.
Autoplay autoplay If specified, the video automatically begins to play back as soon as it can do so without stopping to finish loading the data.
Controls controls If this attribute is present, the browser will offer controls to allow the user to control video playback, including volume, seeking, and pause/resume playback.
Width/Height width or height Width and Height of the video's display area, in pixels.
Embed External Content <embed> Embeds external content at the specified point in the document. This element is deprecated now.
Embed Audio <audio> Used to embed sound content in documents.
Anchor <a> Creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address. Inside the opening tag, add additional attributes like href to specify the links destination.
Hyperlink Reference href The URL that the hyperlink points to.
Target target Where to display the linked URL. For example, target= _blank usually opens the link in a new tab.
Table Head <thead> Defines a set of rows defining the head of the columns of the table.
Table Body <tbody> Encapsulates a set of table rows, indicating that they comprise the body of the table.
Table Foot <tfoot> Defines a set of rows summarizing the columns of the table.
Table <table> Creates a table.
Table Header <th> Adds table headings clarify the meaning of data.
Table Row <tr> Adds rows to a table.
Column Span colspan This attribute contains a non-negative integer value that indicates for how many columns the cell extends.
Row Span rowspan This attribute contains a non-negative integer value that indicates for how many rows the cell extends.
Style Information <style> Contains style information for a document, or part of a document.
External Resource Link <link> Specifies relationships between the current document and an external resource. This element is most commonly used to link to stylesheets.
